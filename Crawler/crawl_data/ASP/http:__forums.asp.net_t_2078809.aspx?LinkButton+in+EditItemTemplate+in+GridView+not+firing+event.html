    Handle RowEditing event event to turn the GridView into edit mode:

    protected void GridViewEmployee_RowEditing(object sender, GridViewEditEventArgs e){        GridViewEmployee.EditIndex = e.NewEditIndex; // turn to edit mode         // Rebind GridView to show the data in edit mode}

    To turn back to normal mode you need to set the EditIndex to -1 and then rebind your grid again.

    GridViewEmployee.EditIndex = -1; //swicth back to default mode// Rebind GridView here to show the data in default mode
