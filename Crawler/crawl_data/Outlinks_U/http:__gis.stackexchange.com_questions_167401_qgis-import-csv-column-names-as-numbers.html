QGIS - Import CSV column names as numbers

up vote 4 down vote favorite

I want to import a CSV table (no geometry) to QGIS. The column names which are numbers (e.g. 2004) are not getting imported.

This is my test CSV file:

Name;Eigenschaft;2004;testderName;1;21;21sdfsfaf;11;11;15

This is how it looks in QGIS:

The third column is not called 2004 like I want but field_3

How to solve this?
