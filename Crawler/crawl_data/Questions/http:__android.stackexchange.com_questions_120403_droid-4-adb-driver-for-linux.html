Assuming your device is rooted there is a ppa for adb and fastboot.

Type:

sudo add-apt-repository ppa:nilarimogardwebupd8sudo apt-get updatesudo apt-get install android-tools-adb android-tools-fastboot

Now you should open a shell/rootshell and type

adb

followed by

adb devices

Then your Phone should show up. To delete the app and odex file type:

adb rm /path/to/apkadb rm /path/to/file

or

adb uninstall [-k] packagename

The option -k keeps cache and data on your phone if you want to.

Then end this process by using 

adb reboot

Cheers
