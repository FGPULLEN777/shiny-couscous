Complete action to open a file using different application “always”

up vote 1 down vote favorite

I'm running a Galaxy Note 3 with stock Android Lollipop. I'm aware how it has worked for years with Android when one can choose an application for a certain type of action or file type, and also choosing to just use it this once it always. And if choosing as always this can then be reverted by clearing the defaults of that app. However the case as in the following image is new to me.

In this case I'd like to use the native "create contact" always. How would I do that?
