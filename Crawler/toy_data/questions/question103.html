Bluetooth File Transfer provides OBEX FTP and OPP for android. It allows you to send a file via bluetooth to an android device or any other devices that implement these bluetooth profiles.

The BT stack on Android phones does not support the file transfer profile. You may be able to add this by rooting your phone and loading a custom BT stack.
