up vote 2 down vote favorite

Can it be done? I'd like to be able to set clock limits based on a number of conditions, such as battery temperature, charging/not charging, and what radios are turned on. I'm aware that there are apps for rooted phones that will allow me to do this, but I'm wondering if there are any that don't need my phone to be rooted.
